Current Limitations

Static browser-local draft records only. No encrypted backend storage, no login, no server OCR, no permanent evidence vault, no external filing/submission, and no professional reliance without review.

Production Roadmap

Next build layer: authentication, encrypted file storage, private database, OCR/PDF/DOC parser, audit logs, backups, access controls, reviewer workflow, and security/professional review.

Project Separation

This is the EONS compliance sister project. ZORTEX media/production tools are separate and should remain in the ZORTEX repository.

Owner Rule

EONS organizes evidence and review packets. It does not create final tax advice, approve claims, submit forms, or replace CPA/EA/tax-attorney review.
