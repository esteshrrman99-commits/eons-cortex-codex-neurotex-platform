SIMPLE USER GUIDE

How to use EONS

This guide explains every common box, field, button, upload area, checklist, and review gate in plain language.

Text boxes

Type or paste facts here. Use plain words. Add who, what, when, amount, source, and what proof you have.

Upload buttons

Use these to select screenshots, PDFs, documents, spreadsheets, images, or other files. In this GitHub Pages version, uploads are read locally in the browser. Permanent storage comes later with the encrypted backend vault.

Checklist previews

These show a draft classification. Read it before relying on it. Missing proof must stay marked missing.

Evidence tiers

Use tiers to sort proof: identity/entity, source documents, service proof, FMV/reasonableness, payment/accrual, provider reporting, and professional review.

Review gates

If the gate says blocked or review required, do not treat the record as ready. A professional review is still required.

Owner rule

This software organizes records. It does not create final tax advice, file claims, submit forms, or replace a CPA, EA, tax attorney, or professional reviewer.
