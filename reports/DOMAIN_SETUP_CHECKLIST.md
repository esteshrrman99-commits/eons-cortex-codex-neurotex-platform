# Domain Setup Checklist

Current status: GitHub Pages staging URL active.
Custom domain status: not connected yet.

When ready:
1. Buy or choose domain.
2. Create docs/CNAME with the domain only.
3. In GitHub Settings > Pages, enter the custom domain.
4. Add DNS records at the domain provider.
5. Wait for DNS/HTTPS verification.

Do not add a domain until owner approves.
