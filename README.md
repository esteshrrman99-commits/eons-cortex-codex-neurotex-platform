# EONS Compliance Engineering Platform Starter

GitHub Pages-ready static platform shell for an owner-only evidence-gated compliance engineering dashboard.

Public Pages output is in `docs/`. Private source packs, evidence vault drafts, intake records, and IP packets stay private/local and should not be exposed publicly.

Safety boundary: this system organizes evidence and review packets. It does not file tax returns, create credits, guarantee outcomes, or replace CPA/EA/tax-attorney review.
