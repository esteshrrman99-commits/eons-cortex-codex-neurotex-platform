Why Not GitHub-Only?

GitHub Pages is excellent for:
- static UI
- public route map
- proof reports
- control center
- non-sensitive docs

GitHub Pages is not enough for:
- private database
- runtime secrets
- server-side encryption
- sensitive file vault
- authenticated upload API
- private audit log

Cloudflare Alternative

Cloudflare Workers + R2 can be used later for:
- edge API
- object storage
- custom upload worker
- R2 private bucket

But it needs a database/auth pairing:
- Supabase Postgres/Auth
- Neon Postgres + auth provider
- D1/KV only if the data model is simplified

Recommended now:
Use Supabase first because it combines auth, database, and storage in one setup path.
