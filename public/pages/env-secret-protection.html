Runtime Secret Rule

Real secrets belong only in:
- backend provider runtime environment
- provider secret manager
- GitHub Actions secrets for deployment automation only

Real secrets must not go in:
- Git
- public/
- docs/
- GitHub Pages JavaScript
- screenshots
- chat
- reports/
