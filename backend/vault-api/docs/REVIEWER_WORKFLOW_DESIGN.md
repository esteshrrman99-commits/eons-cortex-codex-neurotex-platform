# EONS Reviewer Workflow Design

Status: scaffold only / disabled by default.

Reviewer types:
- CPA
- EA
- tax attorney
- bookkeeper
- owner pre-review

Workflow:
1. Select matter
2. Select evidence record
3. Add reviewer note
4. Mark missing proof
5. Mark review status
6. Export review packet
7. Professional signoff remains separate and required

Boundary:
No final approval is enabled. No tax/legal/professional reliance is created by this scaffold.
