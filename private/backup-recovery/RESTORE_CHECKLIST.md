# EONS Restore Checklist

Before production restore is allowed:
1. Confirm owner approval
2. Verify archive ID
3. Verify archive hash
4. Verify access role
5. Log restore request
6. Restore into safe review environment first
7. Verify record counts
8. Verify file hashes
9. Document missing/corrupt records
10. Complete security review
11. Complete professional review if needed
