# Universal Upload Intake Layer

Status: static browser-local import.

Supported now:
- screenshots/images: local preview metadata
- text/csv/json/html/md: local text preview
- pdf/doc/docx/xlsx/other: metadata plus best-effort text preview when browser allows

Not enabled yet:
- permanent encrypted storage
- server OCR
- full PDF parser
- full Word/Excel parser
- automatic legal/tax reliance

Professional review remains required.
