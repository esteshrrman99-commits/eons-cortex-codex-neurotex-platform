# EONS Key Management Checklist

Production key management must include:

1. No keys in Git
2. No keys in ChatGPT messages
3. No keys in public GitHub Pages
4. Environment secrets stored only in approved secret manager
5. Key rotation plan
6. Emergency revoke plan
7. Per-file encryption design
8. Envelope encryption design
9. Access log for decrypt/read/export
10. Backup recovery procedure
11. Owner approval before production enablement
12. Security review before sensitive records
